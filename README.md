# Kubernetes Failure Lab

This project intentionally creates Kubernetes failures and documents debugging process.

## Goals

- Master real-world Kubernetes debugging
- Understand root causes behind failures
- Build production-level troubleshooting confidence

## Failures Covered

- CrashLoopBackOff
- ImagePullBackOff
- Service misconfiguration
- Ingress 503
- OOMKilled
- Pending Pods
